GigBuddy

GigBuddy is an freelancer expense reporting mobile application developed for the University of Michigan - Dearborn Capstone project. I lead development of this project with a team of four University of Michigan students. Agile development was employed for the duration of this project as many changes and adjustments needed to be factored in before project completion.



Gigbuddy is a mobile application available on the Apple App Store and Google Play Store for Freelance workers to record finances for gig work. I also put significant work into an admin webpage for this project which allows the Admin to change advertisements within the mobile application.



This application was developed with React-native, Expo, Javascript, JQuery, and is the Admin Webpage is hosted on Heroku.
